# Smart Legal Document Manager - Dashboard

Frontend React application built with Vite and TypeScript.

## Setup
1. `npm install`
2. `cp .env.example .env`
3. `npm run dev`

Runs on port 5173 by default. Ensure the backend FastAPI server is running on `http://localhost:8000`, and keep `VITE_API_BASE_URL=http://localhost:8000/documents` in `frontend/.env` for the end-to-end compare flow.
